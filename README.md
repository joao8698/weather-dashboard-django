# Django Weather Dashboard

Web application developed with Django for real-time weather search using external API integration.

## Features

* City search
* Current temperature
* Air humidity
* Wind speed
* Weather conditions
* Responsive interface

## Technologies

* Python
* Django
* HTML/CSS
* JavaScript
* Bootstrap
* REST API
